## 3️⃣ Render 账号注册（部署后端时需要）

### 📌 用途
免费部署 Node.js 后端 API

### 📝 注册步骤

#### Step 1: 访问 Render 官网
```
🌐 网址: https://render.com
```

#### Step 2: 注册账号
1. 点击右上角 **"Get Started"** 或 **"Sign Up"**
2. 选择注册方式：
   - ✅ **推荐**: 使用 GitHub 账号登录（一键授权）
   - 或使用邮箱注册

#### Step 3: 使用 GitHub 注册（推荐）
1. 点击 **"Sign up with GitHub"**
2. 授权 Render 访问你的 GitHub 账号
3. 完成！无需额外验证

#### Step 4: 使用邮箱注册
1. 输入邮箱地址
2. 设置密码（至少 8 位）
3. 点击 **"Sign Up"**
4. 检查邮箱，点击验证链接
5. 完成验证

### 💰 费用
- **免费套餐**: 
  - ✅ 750 小时/月（足够一个应用全天运行）
  - ⚠️ 15分钟无活动会休眠
  - ⚠️ 休眠后首次访问需要 30-60 秒启动

### 📋 需要准备的信息
- GitHub 账号（推荐）或邮箱
- 项目的 GitHub 仓库地址（需要先上传代码到 GitHub）

## 📞 常见问题

### Q1: 我已经有 GitHub 账号了，还需要注册吗？
**A:** 不需要！直接使用现有账号即可。

### Q2: Render 和 Vercel 必须用 GitHub 登录吗？
**A:** 不是必须，但强烈推荐！使用 GitHub 登录可以：
- 一键授权，无需验证邮箱
- 自动同步仓库
- 代码更新后自动重新部署

### Q3: 免费套餐够用吗？
**A:** 对于学习和演示项目完全够用！
- Render: 750 小时/月（一个应用全天运行）
- Vercel: 无限流量，无休眠

### Q4: 注册需要信用卡吗？
**A:** 不需要！所有平台的免费套餐都不需要绑定信用卡。

### Q5: 如果我不想部署，还需要注册这些账号吗？
**A:** 不需要！如果只是本地开发和测试，MongoDB Atlas 和测试账号已经足够。

### Q6: 密码忘记了怎么办？
**A:** 所有平台都支持"忘记密码"功能，通过邮箱重置即可。

## 🔐 安全建议

### ✅ 密码管理
- 使用强密码（至少 12 位，包含大小写字母、数字、符号）
- 不要在多个平台使用相同密码
- 建议使用密码管理器（如 1Password, LastPass, Bitwarden）

### ✅ 两步验证
- GitHub、Render、Vercel 都支持两步验证（2FA）
- 建议在注册后立即启用

### ✅ 环境变量保护
- 永远不要把密码、密钥提交到 GitHub
- 使用 `.env` 文件存储敏感信息
- 确保 `.gitignore` 包含 `.env`
