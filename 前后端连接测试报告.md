## ❓ 常见问题排查

### Q1: 前端显示 "Loading..." 一直不消失

**可能原因**:
1. 后端服务器未运行
2. API 地址配置错误
3. CORS 错误

**排查步骤**:
1. 检查后端是否运行: 访问 http://localhost:3000/api/projects
2. 打开浏览器控制台查看错误信息
3. 检查 Network 标签是否有失败的请求

---

### Q2: 浏览器控制台显示 CORS 错误

**错误示例**:
```
Access to fetch at 'http://localhost:3000/api/projects' from origin 'http://localhost:5174' 
has been blocked by CORS policy
```

**解决方法**:
1. 确认后端 `server.js` 中有 `app.use(cors())`
2. 重启后端服务器
3. 清除浏览器缓存

---

### Q3: 前端显示 "Failed to fetch"

**可能原因**:
1. 后端服务器未运行
2. 端口号错误

**解决方法**:
1. 检查后端是否在 3000 端口运行
2. 检查 `.env` 文件中的 `VITE_API_URL`
3. 重启前端服务器（Ctrl+C 然后 `npm run dev`）

---

### Q4: 登录后立即退出

**可能原因**:
1. JWT token 验证失败
2. localStorage 被清除

**解决方法**:
1. 打开浏览器开发者工具 > Application > Local Storage
2. 检查是否有 `token` 和 `user` 数据
3. 如果没有，重新登录
4. 检查后端 JWT_SECRET 是否配置正确
